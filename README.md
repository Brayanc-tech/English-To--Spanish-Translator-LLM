# Brayanc-tech Spanish-English LLM Project

## Repository

GitHub: https://github.com/Brayanc-tech/Spanish-to-English-Translator-LLM-

Credit to Raschka, Sebastian. *Build A Large Language Model (From Scratch)*. Manning, 2024. ISBN: 978-1633437166.

Source reference: https://github.com/rasbt/LLMs-from-scratch

Chapter 7 from the GitHub project helped prepare a dataset for supervised fine-tuning of a language model properly from scratch.

This specific LLM focuses on translating English sentences into Spanish with a Puerto Rican dialect. The dataset consists of pairs of English sentences and their corresponding Spanish translations.

## Package Versions
The code uses the following packages:

- `numpy`: PyTorch and TensorFlow dependency
- `matplotlib`: plotting library
- `tiktoken`: tokenizer
- `torch`: deep learning library
- `tqdm`: progress bar
- `tensorflow`: used for OpenAI's pretrained weights
- `llms-from-scratch`: used to help build gpt2 model


Package versions used during development:

- `numpy`: 2.4.4
- `matplotlib`: 3.10.9
- `tiktoken`: 0.12.0
- `torch`: 2.5.1+cpu
- `tqdm`: 4.67.3
- `tensorflow`: 2.21.0
- `llms-from-scratch`: used to help build gpt2 model
